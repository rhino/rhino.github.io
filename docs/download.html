Rhino Downloads

Rhino is available for download both in source and compiled form.

Binaries

You can download binaries (JAR files) from ftp://ftp.mozilla.org/pub/js/. Rhino 1.5 Release 2 is the last qualified release. It is also possible to download the latest rhino build that reflects newer features and bug fixes, but has not been fully qualified. These zip files also include the source.

If you are looking for js.jar for XSLT or for IBM's Bean Scripting Framework (BSF), download one of the zip files below and unzip it.

  • Rhino 1.5R2.
    • Description of changes from 1.5R1.
  • Rhino 1.5R1.
    • Description of changes from 1.4R3.
  • Rhino 1.4 Release 3.
  • Latest Rhino builds.
    • Description of changes from 1.5R2.

Source

The source code for Rhino is available under NPL 1.1. In addition to getting the source from the zip files above, the source code for Rhino can be found in the CVS tree at mozilla/js/rhino. See  source code via cvs for details on how to set up CVS, define your CVSROOT, and login. Once you've done that, just execute the command 
    cvs co mozilla/js/rhino
to get the tip source.

The current tip can also be viewed using LXR at http://lxr.mozilla.org/mozilla/source/js/rhino/.
